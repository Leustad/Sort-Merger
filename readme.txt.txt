# Sorted File Merger

Sort merge delimited files without loading to the memory via desired key

### Example

Files are sorted buy their second column already

#### file #1

zulu	alpha
lima	charlie
oscar	foxtrot

#### file #2

golf	bravo
mike	echo
hotel	whiskey

#### Output

Output is sorted but (A -> Z) by the second column

zulu	alpha
golf	bravo
lima	charlie
mike	echo
oscar	foxtrot
hotel	whiskey
